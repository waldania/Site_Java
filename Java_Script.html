Perguntas sobre JavaScript

  1. JavaScript e Java são a mesma coisa?

    Apesar dos nomes parecidos, JavaScript e Java são diferentes.

    Java é uma linguagem de programação usada para construir aplicações grandes e complexas e é uma linguagem bem variada em seu uso.

    Já o JavaScript é voltado para a web e é usado principalmente para tornar as páginas dinâmicas, permitindo que elas interajam com o usuário de forma mais interessante.

  2. Quando surgiu o JavaScript?

    O JavaScript surgiu em 1995, foi criado pelo Brendan Eich enquanto ele trabalhava na empresa Netscape.

    A ideia era dar mais interatividade aos sites, permitindo que eles fizessem coisas como validar formulários ou alterar conteúdo sem precisar recarregar a página inteira.

  3. Quem foi o responsável pela criação do JavaScript?

    Quem criou o JavaScript foi o Brendan Eich, um programador da Netscape, que a desenvolveu em apenas 10 dias.

    Atualmente, o JavaScript se tornou uma das linguagens mais usadas no mundo, por possibilitar uma infinidade de coisas dentro do desenvolvimento web.

  4. Qual o caminho trilhado pelo responsável pela criação do JavaScript, trata-se de alguma empresa ainda no mercado?

    Depois de criar o JavaScript, o Brendan seguiu uma carreira bem interessante.

    Ele ajudou a fundar a Mozilla, a organização por trás do navegador Firefox, que ainda está ativa.

    Mais tarde, ele também criou a Brave Software, uma empresa que desenvolve o navegador Brave, que tem como foco a privacidade e a segurança do usuário enquanto navega pela web.
